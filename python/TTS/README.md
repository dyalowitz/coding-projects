#README

This is for a rudimentary text-to-speech (TTS) program written in python.

This folder includes instructions, dependencies, tools I used.
In addition to the full program, there is also a separate file for each component of the code ran for to test each component.

  1. Engines and voices
  2. Core systhesis to file
  3. GUI shell with stubbed synthesis

#Tools used:
  1. Python
  2. Notepad++

#Dependencies:
  1. pyttsx3

#Instructions:
1. Install Python - https://www.python.org/downloads/
2. Install Notepad++ - https://notepad-plus-plus.org/downloads/
3. Install the dependencies
   - `pip install pyttsx3` or `python -m pip install pyttsx3`
5. Run the tests in order.
   - Engine-and-voices.py
   - Core-synthesis-to-file.py
   - gui-shell-with-stubbed-synthesis.py
6. Once the tests run as intended and expected, run the full program.
   - tts.py
